Orbital Element Methods
=======================

.. method:: double ArgumentOfPeriapsis()

   Return the argument of periapsis of platform's orbit in degrees.

.. method:: double Eccentricity()

   Return the eccentricity of the platform's orbit.

   .. note:: The eccentricity is equal to zero for a circular orbit and approaches 1 for a nearly parabolic orbit.

.. method:: double Inclination()

   Return the inclination of platform's orbit in degrees.

.. method:: double MeanAnomaly()

   Return the mean anomaly of the platform's orbit at the current time.

.. method:: double MeanMotion()

   Return the mean motion of the platform's orbit in degrees per second.

.. method:: double RAAN()

   Return the right ascension of the ascending node of a platform's orbit in degrees.

.. method:: double RevolutionsPerDay()

   Return the number of revolutions per day of the platform's orbit.

.. method:: double SemiMajorAxis()

   Return the semi-major axis of the platform's orbit in meters.

.. method:: double PeriapsisRadius()
            double Periapsis()

   Return the periapsis radius of the platform's orbit in meters.

   .. note:: The `Periapsis` method is deprecated and will be removed in a future release. `PeriapsisRadius` is preferred.

.. method:: double PeriapsisAltitude()

   Return the periapsis altitude of the platform's orbit in meters. This altitude is referenced to the mean radius of the :ref:`central_body<central_body_commands>`

.. method:: double ApoapsisRadius()
            double Apoapsis()

   Return the apoapsis radius of the platform's orbit in meters.

   .. note:: The `Apoapsis` method is deprecated and will be removed in a future release. `ApoapsisRadius` is preferred.

.. method:: double ApoapsisAltitude()

   Return the apoapsis altitude of the platform's orbit in meters. This altitude is referenced to the mean radius of the :ref:`central_body<central_body_commands>`.

.. method:: double TrueAnomaly()

   Return the true anomaly of the platform's orbit at the current time.

.. method:: string Descriptor()

   Return the descriptor (tag) for the space vehicle. If a TLE is being used, the descriptor is line 0; otherwise, it is provided in the :command:`WSF_SPACE_MOVER.descriptor` input.

.. method:: string Designator()

   Return the designator for the space vehicle. If a TLE is being used, the designator is the TLE satellite international designator; otherwise, it is either provided in the :command:`WSF_SPACE_MOVER.designator` input, or assigned a default value of "00001A".
