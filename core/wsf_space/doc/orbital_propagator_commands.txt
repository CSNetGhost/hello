Deprecated Orbital Propagator Commands
**************************************
.. block:: deprecated_orbital_propagator_commands

.. command:: egm_96
.. command:: wgs_84
.. command:: heliocentric

   Specify use of the wgs_84 gravitational parameter, the egm_96 gravitational parameter, or the heliocentric
   gravitational parameter for propagation.  The **heliocentric** option is used to simulate bodies orbiting the
   sun.

   .. note::
      This setting is only effective for :model:`WSF_SPACE_MOVER`.

   **Default** wgs_84

.. deprecated:: 2.9
   The following debug output commands will be removed in a future release.

.. command:: debug_output_wsf <boolean-value>

   Write additional information to standard output that can be processed by the CME tool "sedit'.

.. command:: debug_output_oe <boolean-value>

   Write additional information about the orbital elements to standard output.

.. command:: debug_output_stk <boolean-value>

   Write additional information to standard output that can be used for comparing the orbit with the Systems Tool Kit
   (STK) from Analytical Graphics, Inc.

.. command:: debug_output_xyz <boolean-value>

   Write additional information to standard output that can be used for comparing the orbit with the Systems Tool Kit
   (STK) from Analytical Graphics, Inc.
