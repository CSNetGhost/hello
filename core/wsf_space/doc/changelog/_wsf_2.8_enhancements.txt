* **(2.8.2)** Added variant of :method:`WsfSpaceMover.SetOrbit` that accepts an initial epoch as a parameter. (:issue:`AFSIM-1254`) 
* **(2.8.1)** Added methods to :class:`WsfSpaceMover` to compute common space mission-relevant geometric quantities (:method:`WsfSpaceMover.LookAngleTo`, :method:`WsfSpaceMover.IncidenceAngle`, :method:`WsfSpaceMover.TargetElevationAngle`, :method:`WsfSpaceMover.SquintAngleTo`, :method:`WsfSpaceMover.DistanceToHorizon`, and :method:`WsfSpaceMover.LookAngleToHorizon`). (:issue:`AFSIM-1152`)
* **(2.8.1)** Added :method:`WsfSpaceMover.InitialHeading`, which calculates the heading of a satellite from its initial velocity state. (:issue:`AFSIM-697`)
* Added versions of :method:`WsfSpaceMover.ComputeRendezvous` and :method:`WsfSpaceMover.ComputeIntercept` that calculate a rendezvous and intercept with a given fixed transfer time. (:issue:`AFSIM-973`)
* Added :method:`RIC<WsfSpaceMover.RIC>` and :method:`RIC_AtTime<WsfSpaceMover.RIC_AtTime>` functions to :class:`WsfSpaceMover`. (:issue:`AFSIM-536`)
* Added :ref:`orbital_dynamics_terms.atmospheric_drag` to the :model:`WSF_INTEGRATING_SPACE_MOVER`, and associated script class :class:`WsfAtmosphericDragTerm`. (:issue:`AFSIM-535`)
* Added the :class:`WsfAtmosphere`, :class:`WsfPiecewiseExponentialAtmosphere`, and :class:`WsfJacchiaRobertsAtmosphere` script classes. (:issue:`AFSIM-535`)
* Added options to specify and access offsets for targeting-class maneuvers in the Radial, In-track, Cross-track (RIC) frame (see :command:`target` and :method:`WsfTargetOffsetFrame.RIC`). (:issue:`AFSIM-659`)
* Added :command:`WSF_SPACE_MOVER.periapsis_radius`, :command:`WSF_SPACE_MOVER.periapsis_altitude`, :command:`WSF_SPACE_MOVER.apoapsis_radius`, :command:`WSF_SPACE_MOVER.apoapsis_altitude` as available orbital element commands for space movers. (:issue:`AFSIM-517`)
* Added the :command:`WSF_SPACE_OPTICAL_SIGNATURE.high_resolution_eclipse` option to the :model:`WSF_SPACE_OPTICAL_SIGNATURE` to perform more accurate computations of the illumination state of the platform. (:issue:`AFSIM-527`)
* Added option to specify delta-v for :doc:`../delta_v` orbital maneuvers in the Radial, In-track, Cross-track (RIC) frame. (:issue:`AFSIM-748`)
* Added options to specify relative velocity offsets in :command:`target` input blocks and the :class:`WsfTargetPoint` class. (:issue:`AFSIM-749`)
* Added a :doc:`../natural_motion_circumnavigation` orbital maneuver with corresponding class :class:`WsfNaturalMotionCircumnavigation`. (:issue:`AFSIM-749`)
