* Removed the previously deprecated target specification options, platform and offset, from the :doc:`../target`, :doc:`../intercept`, and :doc:`../rendezvous` maneuvers. (:issue:`AFSIM-693`)
* Removed previously deprecated methods from the script classes :class:`WsfTargetManeuver`, :class:`WsfInterceptManeuver`, and :class:`WsfRendezvousManeuver` that constructed the maneuver or modified the maneuver's target information without using the :class:`WsfTargetPoint` . (:issue:`AFSIM-693`)
* Deprecated the versions of :method:`WsfSpaceMover.ComputeRendezvous`, and :method:`WsfSpaceMover.ComputeIntercept` that accept a :class:`WsfTrackId` as an argument. Users should instead use the versions accepting a :class:`WsfTargetPoint` instance. (:issue:`AFSIM-692`)
* Deprecated orbital propagator debug output commands :command:`deprecated_orbital_propagator_commands.debug_output_wsf`, :command:`deprecated_orbital_propagator_commands.debug_output_oe`, :command:`deprecated_orbital_propagator_commands.debug_output_stk`, :command:`deprecated_orbital_propagator_commands.debug_output_xyz`, and debug_output_tle. (:issue:`AFSIM-694`)
* Clarified the documentation with respect to hyperbolic end-states of maneuvers (e.g., :doc:`../rendezvous`), and other orbital targeting operations (e.g., :method:`WsfSpaceMover.ComputeIntercept`). (:issue:`AFSIM-745`)
