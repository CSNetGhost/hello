* Added :ref:`ECLIPSE_ENTRY` and :ref:`ECLIPSE_EXIT` events to :command:`event_output`, :command:`csv_event_output`, and :command:`observer`. *(Issue #424)*
* When retrieving a :ref:`two line element <docs/wsf_space_mover:Orbital Element Commands>` from a :model:`WSF_NORAD_SPACE_MOVER`, the :method:`WsfSpaceMover.TwoLineElement` method now returns the correct averaged orbital elements in the :ref:`two line element <docs/wsf_space_mover:Orbital Element Commands>`. (previously, only the instantaneous orbital elements were returned).  This enhancement allows for proper initialization of another :model:`WSF_NORAD_SPACE_MOVER`, either through input or with the :method:`WsfSpaceMover.SetTwoLineElement` script method. *(Issue #1014)*
* New script class :class:`WsfSolarTerminator` provides utilities related to the Solar terminator. *(ELSZ #183)*
* Added :method:`WsfSpaceMover.Descriptor` and :method:`WsfSpaceMover.Designator` methods to provide script access to the :command:`WSF_SPACE_MOVER descriptor <WSF_SPACE_MOVER.descriptor>`. *(Issue #1180)*
* Added a :doc:`DeltaV_Maneuver<../delta_v>` maneuver with a corresponding :class:`WsfDeltaV_Maneuver` script class. *(Issue #1183)*
