* **(Issue #384)** Added a new :command:`optical_signature` model :model:`WSF_SPACE_OPTICAL_SIGNATURE` which provides a dynamic infrared and visual signature based on the position of the platform, Sun and Earth.
* **(Issue #420)** Added a :ref:`script interface<WSF_SPACE_MOVER.script_interface>` to :model:`WSF_SPACE_MOVER`, and provided a new scripted :ref:`orbital mission event<WSF_SPACE_MOVER.orbital_mission_sequence>` capability (both :doc:`input<../scripted>` and :class:`scripted<WsfOrbitalScriptEvent>` versions).
* **(Issue #498)** Added the :model:`WSF_ORBIT_DETERMINATION_FILTER` to provide more refined orbit determination solutions.  It has been integrated for use directly with :doc:`../orbit_determination_fusion`, or it can be used in a :command:`filter` definition.
* **(Issue #570)** Added :command:`observer` script interface for ORBIT_DETERMINATION_INITIATED and ORBIT_DETERMINATION_UPDATED.
* **(Issue #642)** Added new methods to :class:`WsfSpaceMover` to align spacecraft orientation relative to an :class:`WsfArticulatedPart` or equivalent Euler angles that define a :ref:`PCS<PCS>` angular offset.
