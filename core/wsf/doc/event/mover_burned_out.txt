MOVER_BURNED_OUT
----------------

Provides a notification that a rocket powered mover has exhausted thrust.

event_output Signature:

.. parsed-literal::

   <time> <event> <platform>

Signature Elements:

======================================== ========================================
Field                                    Description
======================================== ========================================
<time>                                   The current simulation time
<event>                                  The event name
<platform>                               Name of platform affected
======================================== ========================================
