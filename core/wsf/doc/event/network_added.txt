NETWORK_ADDED
-------------

event_output Signature:

.. parsed-literal::

   <time> <event> <network> <address>

Signature Elements:

======================================== ========================================
Field                                    Description
======================================== ========================================
<time>                                   The current simulation time
<event>                                  The event name
<network>                                The network object being added to the Network Manager
<address>                                The address of the network being added to the Network Manager
======================================== ========================================
