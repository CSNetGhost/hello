* **(Issue #493)** Added the :method:`Calendar.TimeSince` in script :class:`Calendar`.
* **(Issue #479)** Added the :ref:`error_model.standard_sensor_error` and :ref:`error_model.radar_sensor_error` sensor error models. Also refactored the simple Doppler model in the :model:`WSF_RADAR_SENSOR` into a :command:`WSF_RADAR_SENSOR.signal_processor` type called a :ref:`signal_processor.simple_doppler`
* Added in :method:`WsfSimulation.ClockRate` to get the current simulation :command:`clock_rate` for realtime simulations and :method:`WsfSimulation.IsRealtime` to see if simulation mode is set to :command:`realtime`.
* **(Issue #547)** Added the script class :class:`Format` and the method :method:`string.Pad` which allows one to control the output format and precision of floating point data and to pad output fields to a specified width. You can select fixed, scientific, latitude, longitude or time format.
* **(Issue #576)** Added the script methods :method:`WsfFuel.DistanceAtBingo`, :method:`WsfFuel.DistanceAtReserve`, :method:`WsfFuel.DistanceToBingo`, :method:`WsfFuel.DistanceToReserve`, :method:`WsfFuel.DistanceToEmpty`, :method:`WsfFuel.TimeAtBingo`, and :method:`WsfFuel.TimeAtReserve` to class :class:`WsfFuel` to get the estimated distances/times that the platform can travel for different fuel states.
* Added unit conversion static methods :method:`Math.KG_PER_LB` and :method:`Math.NT_PER_LB`
