* **(2.8.3)** Removed platform state data from :command:`event_output` and :command:`csv_event_output` for platforms that have not been initialized. (:issue:`AFSIM-1878`)
* **(2.8.1)** Updated documentation of :class:`WsfPlatform` script methods for setting location, velocity, and orientation to caution usage of these methods on uninitialized platforms created via script. (:issue:`AFSIM-190`)
* Added boolean inputs, defaulted to true, and deprecated the :command:`dis_interface` commands :command:`dis_interface.suppress_cme_detect_beam`, :command:`dis_interface.suppress_cme_entity_data`, :command:`dis_interface.suppress_cme_passive_sensor`, and :command:`dis_interface.suppress_cme_draw_data`. Marked :command:`dis_interface.use_deprecated_cme_draw_data` as deprecated. (:issue:`AFSIM-714`)
* Removed several undocumented and deprecated script methods from :class:`WsfPlatform` which provided simulation management facilities already present as static methods in :class:`WsfSimulation`, including ``FindPlatform`` and ``DeletePlatform``. (:issue:`AFSIM-756`)
* Modifications to the simulation initialization sequence with regard to platforms may produce slightly different simulation outcomes for regression testing. (:issue:`AFSIM-543`)
