Military
^^^^^^^^

* :command:`acoustic_signature` - Define the acoustic signature of a platform
* :command:`infrared_signature` - Define the infrared signature of a platform.
* :command:`inherent_contrast` - Define the inherent contrast of a platform.
* :command:`optical_reflectivity` - Define the optical reflectivity for a platform.
* :command:`optical_signature` - Define the optical signature (projected area) for a platform.

Electronic Warfare
^^^^^^^^^^^^^^^^^^

:ref:`Electronic_Warfare_Effect_Aggregation` for understanding how EW is aggregated and used in sensors and communication receivers.

* :command:`electronic_warfare` - Define the Electronic Warfare object types. (:ref:`Predefined_Electronic_Warfare_Types`)
* :command:`electronic_warfare_effect` - Define the Electronic Warfare effect object types. (:ref:`Predefined_Electronic_Warfare_Effect_Types`)
* :command:`electronic_warfare_technique` - Define the Electronic Warfare technique object types. (:ref:`Predefined_Electronic_Warfare_Technique_Types`)
* :command:`false_target` - Define a false target technique for electronic attack.
* :command:`false_target_screener` - Define the response of a receiver to false targets.
