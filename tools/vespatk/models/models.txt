# model definition file
#
# model definitions:
# "model name
#    filename address_of_modelfile
# end_model"
#
# modelfiles may be ive, osgb, flt, obj, or any model natively supported by OSG
#
# the model definition may include a pre_xform block which can contain transformations:
# "scale value"
# "translate valueX valueY valueZ"
# "rotate axisChar(x,y,or z) valueAngleDeg"
#
# the transformations will be applied in the order they appear
# it is preferable to transform the models themselves
# 
# the model definition may include:
# "screen_scale value"
# "wing_tip valueX valueY valueZ" (denotes right_tip after all transformations, left_tip should be reflection over x-z plane)
# "set immersive" (indicates that this model should be used in immersive views)
# "default_set" (indicates a model with a defined set should also be used in the default set)
# "articulation visual_part node" (maps a visual-part from the simulation to a node on the 3d-model for articulation)
#
# aliases
# "alias alias_name model_name"
#
# applications may reference the model name or alias directly
